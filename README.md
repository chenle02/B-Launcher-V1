# B-Launcher V1

Animated blue magnifying glass launcher for B apps and phone apps.

Features:
- Animated blue magnifying glass
- Search bar
- Shows B-Map, B-Notes, and B-Translate first
- Scans launchable apps installed on the phone
- Search every phone app
- Opens apps directly
- Reload Apps button
- Dark blue B-style UI
- GitHub Actions APK build
